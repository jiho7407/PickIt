## 전체 구조

```
고민 생성 → 참여 유도 → 결과 확인 → 행동 유도 → 7일 후 회고 → 리텐션
```

## 상황별 알림 문구

### 고민 올린 직후 (~30분)

```
🔥 **당신의 고민에 사람들이 반응하고 있어요
[글 제목 : ex. 러닝할 때 신으려고 하는데 살까요 말까요 or 상품명: ex. 호카 카본 X3]에 7명이 투표했습니다**
```

```
😮 생각보다 반응 갈리고 있어요
지금 52% vs 48%로 의견이 나뉘었어요
```

### 투표 급증 시

```
🔥 지금 23명이 당신 고민 보고 있어요
결과 확인하러 가기 →
```

### 방치된 고민 (24시간 후)

```
⏳ 아직 결정 못 하셨나요?
지금 사람들이 뭐라고 했는지 확인해보세요
```

### 구매/절제 행동 유도

```
🚫 **78%가 "사지 말자"에 투표했어요
지금 참으면 148,720원 아낄 수 있어요**
```

```
✅ **81%가 "사도 된다"고 했어요
지금이 가장 확신 있는 타이밍이에요**
```

### 7일 후 팔로업 ⭐️

```
📌 **그때 그 고민, 결국 어떻게 하셨어요?
결과만 알려주시면 절약 금액이 계산됩니다**
```

```
👀 그때 그 고민… 기억나시죠?
결국 사셨나요, 참으셨나요?
```

### 절약 성공 시

```
🎉 잘 참았어요
148,720원 아꼈습니다
```

```
🔥 3회 연속 절약 성공
습관이 만들어지고 있어요
```

### 재방문 유도 (주 1회)

```
💸 **오늘 가장 핫한 고민
"취준생인데 에어팟 프로 사도 될까요?"**
```

```
💬 당신이 도와줄 차례입니다
누군가 고민 중이에요
```

### 이탈 방지 (2주 이상 미접속)

```
📉 요즘 소비 괜찮으세요?
최근 소비 기록이 멈췄어요
```

```
👀 당신이 사라진 사이
128개의 고민이 올라왔어요
```

### 카카오톡 메시지로 보내야 할 알림 (정말 필요한 알림만, 차단 당하는 일 없게)

- 투표 결과 나왔을 때
- 7일이 지나고 팔로우업해야 할 때(올린 상품 실구매했는지 or 안했는지)
- 비슷한 상황 사람 고민 올라왔을 때(비슷한 연령, 나이대 사람 → 사이트 유입해서 투표 독려
  - `20대 취준생`이면 → 비슷한 상황 사람 고민 우선 푸시
  - 자주 흔들리는 카테고리 → 해당 고민 알림 우선
  > "당신과 비슷한 상황의 누군가가 고민 올렸어요"
  >
  > 일반 알림보다 클릭률 증가 예상

#### 처음에 로그인 기능 구현할 건지?

**알림 허용 타이밍이 핵심**

- 처음 들어오자마자 → 거절할 확률 높음
- 고민 올리고 나서 → <투표 결과 알려드릴게요, 알림 허용하시겠어요?> 창 노출(이때 연락처 받기)

→ 로그인은 고민글 올릴 때 로그인 유도, 투표는 로그인 없이도 가능하게
